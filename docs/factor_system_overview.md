# SmartTrader 交易因子系统

## 项目概述

SmartTrader 因子系统是量化交易的核心模块，通过多维度因子分析为投资决策提供数据支持，结合 AI 大模型实现智能信号生成。

---

## 一、什么是交易因子？

交易因子是能够预测资产未来收益的特征指标。在量化投资中，因子是决策的基础单元，类似于传统投资中的"判断依据"。

### 因子的作用

```
资产数据 → 因子计算 → 因子评分 → 信号生成 → 投资决策
   │           │           │           │           │
 价格/成交量   数学公式    标准化处理   AI综合分析   买入/卖出
```

### 为什么需要多因子？

单一因子存在局限性：
- 动量因子在震荡市失效
- 价值因子在成长牛市跑输
- 情绪因子可能出现误判

**多因子组合** 可以相互验证、降低风险、提高胜率。

### 3.2 因子计算公式

#### 动量因子
```
Momentum = (当前价格 - N日前价格) / N日前价格

评分规则：
- 涨幅 0-50% → 评分 0 到 +1
- 跌幅 0-50% → 评分 0 到 -1
```

#### 波动率因子
```
Volatility = √(Σ(日收益率 - 平均收益率)² / (N-1))

评分规则：
- 低波动 → 正分（风险低）
- 高波动 → 负分（风险高）
```

#### 贝塔因子
```
Beta = Cov(资产收益率, 基准收益率) / Var(基准收益率)

基准选择：
- 加密货币：BTC
- 股票：S&P 500 (SPY)
- 商品：GLD

评分规则：
- Beta ≈ 1 → 正分（与市场同步）
- Beta 偏离1 → 负分（波动异常）
```

#### 成交量比率因子（大资金流入）
```
VR = 上涨日平均成交量 / 下跌日平均成交量

评分规则：
- VR > 1.5 → 正分（大资金流入）
- VR < 0.8 → 负分（资金流出）
- 0.8-1.5 → 中性
```

#### 趋势因子
```
Trend = 线性回归斜率 (N日价格序列)

评分规则：
- 斜率 > 0 → 正分（上升趋势）
- 斜率 < 0 → 负分（下降趋势）
```

#### 5.3.8 因子相关性计算原理

**什么是 Pearson 相关系数？**

衡量两个因子"一起变化"的程度，范围 -1 到 +1。

```
相关性值    含义                    例子
─────────────────────────────────────────────────────
  +1.0     完全正相关              动量高 → 趋势也高
  +0.7     高度正相关 ⚠️           需要调整权重
  +0.3     弱正相关                正常
   0.0     无相关                  两者独立
  -0.3     弱负相关                正常
  -0.7     高度负相关 ⚠️           需要调整权重
  -1.0     完全负相关              动量高 → 波动率低
```

**计算步骤 (简化版)**

```
输入：两个因子在所有资产上的值
  - 因子A (动量): [0.6, 0.4, -0.2, 0.1]  (BTC, ETH, AAPL, GLD)
  - 因子B (趋势): [0.5, 0.3, -0.1, 0.2]

步骤1: 计算平均值
  mean_A = (0.6 + 0.4 - 0.2 + 0.1) / 4 = 0.225
  mean_B = (0.5 + 0.3 - 0.1 + 0.2) / 4 = 0.225

步骤2: 计算分子 (两者一起偏离均值的程度)
  = (0.6-0.225)×(0.5-0.225) + (0.4-0.225)×(0.3-0.225) + ...
  = 0.103 + 0.013 + 0.138 + 0.003
  = 0.257

步骤3: 计算分母 (各自偏离程度的乘积)
  = √[(0.6-0.225)² + (0.4-0.225)² + ...] × √[(0.5-0.225)² + ...]
  = 0.607 × 0.435
  = 0.264

步骤4: 计算相关系数
  = 0.257 / 0.264 = 0.97 (高度正相关)
```

**为什么需要避免高相关性？**

```
问题：动量(20%) + 趋势(20%) 相关性 0.7

实际效果：
  - 两个因子"说同一件事"
  - 相当于给"趋势"因子 40% 权重
  - 其他因子被稀释，决策偏差

解决：
  - 降低其中一个权重 (如 动量 10%, 趋势 10%)
  - 或合并为一个因子
```

**代码对应关系**

```ruby
# app/controllers/admin/factor_definitions_controller.rb

# 计算两个因子序列的相关性
def calculate_pearson_correlation(x, y)
  # 1. 空值检查
  return 0 if x.empty? || y.empty? || x.size != y.size

  # 2. 计算均值
  n = x.size
  mean_x = x.sum / n.to_f
  mean_y = y.sum / n.to_f

  # 3. 计算分子 (协方差)
  numerator = x.each_with_index.sum { |xi, i| (xi - mean_x) * (y[i] - mean_y) }

  # 4. 计算分母 (标准差乘积)
  denominator_x = Math.sqrt(x.sum { |xi| (xi - mean_x) ** 2 })
  denominator_y = Math.sqrt(y.sum { |yi| (yi - mean_y) ** 2 })

  return 0 if denominator_x.zero? || denominator_y.zero?

  # 5. 返回相关系数
  numerator / (denominator_x * denominator_y)
end

# 找出高相关性因子对 (阈值 0.6)
def find_high_correlations(factors, matrix)
  threshold = 0.6

  factors.each_with_index do |factor_a, i|
    factors.each_with_index do |factor_b, j|
      next if i >= j  # 跳过下三角和对称位置，避免重复

      correlation = matrix[factor_a.id][factor_b.id]
      if correlation.abs >= threshold
        # 标记为高相关性，提示用户调整
      end
    end
  end
end
```

## 六、因子评估指标

### 6.1 IC (Information Coefficient)

**定义：** 因子值与未来收益的相关系数

```
IC = Correlation(因子值, 未来N日收益)

判断标准：
- |IC| > 0.05：强因子
- |IC| > 0.03：有效因子
- |IC| < 0.02：弱因子
```

### 6.2 IR (Information Ratio)

**定义：** IC的均值除以IC的标准差

```
IR = Mean(IC) / Std(IC)

判断标准：
- IR > 0.5：优秀因子
- IR > 0.3：良好因子
- IR < 0.1：无效因子
```

### 6.3 胜率

**定义：** 因子预测正确的比例

```
胜率 = 预测正确次数 / 总预测次数

判断标准：
- 胜率 > 55%：有效因子
- 胜率 > 60%：优秀因子
```

---

## 七、风险管理

### 7.1 因子相关性控制

高相关性因子会降低组合效果，需要监控因子间相关性：

```
相关性矩阵示例：

          动量  波动率  成交量  趋势
动量       1.0   -0.3   0.4    0.7
波动率    -0.3    1.0   -0.2   -0.4
成交量     0.4   -0.2    1.0    0.3
趋势       0.7   -0.4    0.3    1.0

注意：动量与趋势相关性较高(0.7)，需考虑降低权重或正交化处理
```

### 7.2 因子衰减监控

因子有效性会随时间衰减，需要持续监控：

```
监控指标：
- IC 月度变化趋势
- IR 季度变化趋势
- 胜率年度对比
```
